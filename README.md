# Contact Me Form

A fully accessible, responsive contact me form built with semantic HTML5 and CSS.

## Project Overview

This is an accessible contact form designed for freelancers and agencies to collect detailed project information from potential clients. The form demonstrates best practices in web accessibility, ensuring it can be used by everyone, including people with disabilities.

## Features

### Accessibility Features

- **Compliant** - Meets international accessibility standards
- **Screen Reader Support** - Properly labeled fields with ARIA attributes
- **Focus Indicators** - Visible focus states on all interactive elements
- **Error Handling** - Clear error messages with icons and distinct color
- **Help Text** - Contextual assistance for complex fields
- **Semantic HTML** - Proper use of fieldsets, legends, and labels

### Form Sections

1. **Personal Information** - Name, email, phone, company
2. **Project Details** - Project type, description, features, deadline
3. **Additional Information** - Timeline, file uploads, referral source
4. **Terms & Conditions** - Agreement checkboxes

### Input Types

- Text inputs with validation
- Email and phone inputs
- Radio button groups (custom styled)
- Checkbox groups (custom styled)
- Dropdown selects
- Textarea
- Date picker
- File upload

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and layout
- **ARIA** - Accessibility attributes
- **Flexbox** - Responsive layouts
- **CSS Custom Properties** - Color theming

## Getting Started

### View Live Demo
[Live Demo](https://ssyevuo.github.io/contact-me/)

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/yourusername/accessible-form.git
```

2. Navigate to the project directory:
```bash
cd accessible-form
```

3. Open `index.html` in your browser:
```bash
# Mac
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

No build process or dependencies required!
